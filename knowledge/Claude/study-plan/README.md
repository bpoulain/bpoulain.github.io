# Plan d'étude — 30 jours

## Comment utiliser ce plan

Ce plan d'étude découpe le bootcamp en sessions quotidiennes actionnables.
Chaque jour a un **objectif**, une **durée estimée**, un **exercice concret** et un **critère de réussite**.

### Choisissez votre rythme

| Rythme | Temps/jour | Durée totale | Profil |
|--------|-----------|--------------|--------|
| Intensif | 2h+ | 4 semaines | Vous avez du temps dédié |
| Standard | 1h | 6 semaines | Usage quotidien régulier |
| Partiel | 30 min | 8-10 semaines | En parallèle d'un emploi chargé |

**Recommandation** : 45-60 min/jour est le sweet spot. En dessous, pas assez de pratique. Au-dessus, la fatigue cognitive réduit la rétention.

### Conseils pour retenir

1. **Pratiquer le jour même** — lire sans pratiquer = 10% de rétention
2. **Un concept par session** — ne pas surcharger
3. **Réutiliser immédiatement** dans un contexte réel (pas juste l'exercice)
4. **Tenir le log** — notez dans `memory/decisions.md` ce que vous apprenez

### Adapter par rôle

Suivez le plan de base (semaine 1), puis basculez sur votre fichier de rôle :
- `roles/bootcamp-pm.md` pour les semaines 2-4
- `roles/bootcamp-builder.md`
- `roles/bootcamp-designer.md`
- `roles/bootcamp-design-engineer.md`

## Comment mesurer vos progrès

Utilisez `self-evaluation.md` à la fin de chaque semaine.

Règle simple : si vous pouvez accomplir la "preuve de maîtrise" de chaque module sans regarder les instructions, vous avez validé.
