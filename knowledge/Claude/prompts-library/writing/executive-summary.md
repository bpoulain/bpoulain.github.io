# Résumé exécutif

**Usage** : Synthétiser un document long (PRD, rapport, analyse) pour un CEO/investisseur  
**Input** : Document à synthétiser  
**Output** : Résumé d'1 page max, orienté décision

---

Tu es expert en communication exécutive.

Crée un résumé exécutif d'1 page maximum pour ce document.

**Audience** : [CHOISIR : CEO / Investisseur / Board / Direction / Équipe complète]  
**Objectif du résumé** : [CHOISIR : Obtenir une validation / Informer / Préparer une décision]  
**Ton** : [CHOISIR : Formel / Direct / Data-driven]

**Structure obligatoire** :

```
## [Titre clair]

### Contexte (2-3 phrases)
[Pourquoi ce document existe]

### Situation actuelle
[1-2 chiffres clés qui montrent le problème ou l'opportunité]

### Recommandation / Décision demandée
[Une phrase : ce qu'on propose de faire]

### Pourquoi maintenant
[2-3 raisons avec données si disponibles]

### Ce qu'on attend de vous
[Action précise : valider / approuver budget / prendre une décision sur X]

### Risques principaux (max 3)
[Ce qui pourrait mal tourner]

### Prochaines étapes si validé
[Chronologie simple : J1-J7, J8-J30]
```

**Règles** :
- Maximum 300 mots
- Chaque phrase doit ajouter de la valeur — pas de remplissage
- Finir par une demande d'action claire

[COLLER LE DOCUMENT ICI]
