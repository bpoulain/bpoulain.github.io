# Analyse concurrentielle approfondie

**Usage** : Analyser un concurrent spécifique en profondeur  
**Input** : Nom du concurrent + informations disponibles (site, G2, Product Hunt, etc.)  
**Output** : Rapport complet avec forces, faiblesses, opportunités

---

Tu es product analyst spécialisé en intelligence concurrentielle B2B SaaS.

Analyse [NOM DU CONCURRENT] pour nos cas d'usage [NOTRE DOMAINE].

**Contexte** : [DÉCRIRE NOTRE PRODUIT EN 2 PHRASES]

**Instructions** :
1. Sois honnête sur leurs forces — ne pas les minimiser
2. Identifie les vrais différenciateurs (pas le marketing)
3. Concentre-toi sur les cas d'usage qui chevauchent les nôtres
4. Chaque opportunité doit être actionnnable (pas vague)
5. Marque "(hypothèse)" ce qui n'est pas vérifié
