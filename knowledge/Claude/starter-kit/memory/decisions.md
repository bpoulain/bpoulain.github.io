# Log des Décisions

> Toute décision produit ou technique significative est loggée ici.
> Format : date, décision, contexte, alternatives rejetées, raison du choix.
> NE PAS supprimer les entrées passées — elles expliquent pourquoi on est où on est.

---

## Template d'entrée

```
### [YYYY-MM-DD] — [Titre de la décision]

**Décision** : [Ce qu'on a décidé]

**Contexte** : [Pourquoi cette décision était nécessaire]

**Alternatives considérées** :
- [Option A] — rejetée parce que [raison]
- [Option B] — rejetée parce que [raison]

**Raison du choix** : [Pourquoi cette option]

**Conséquences attendues** : [Ce que ça implique]

**Owner** : [Qui a pris la décision]
```
